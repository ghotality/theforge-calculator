# The Forge - Crafting Calculator

An advanced web-based calculator for The Forge game that helps players optimize their crafting builds. Calculate the odds of forging different weapons and armor, discover ore traits, and plan your crafting strategy with real-time probability calculations.

## Features

- **Real-time Crafting Calculator**: Input ores and instantly see the probability of crafting different weapon and armor types
- **Ore Trait Analysis**: Discover which traits will be transferred based on your ore composition
- **Multiplier Calculation**: See the combined multiplier effect of your selected ores
- **Comprehensive Ore Database**: Browse all available ores sorted by multiplier with search functionality
- **Responsive Design**: Works seamlessly across desktop, tablet, and mobile devices

## About

This calculator is designed to help players of The Forge game make informed decisions when crafting items. By selecting different ores and quantities, players can see the exact probabilities of obtaining specific weapon or armor types, along with the traits that will be transferred based on ore composition percentages.

The tool is built with Next.js and is hosted on Vercel for easy access without any installation required.
